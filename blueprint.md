# Project Blueprint: Information-Focused Blog Website

## Overview
A modern, framework-less blog-style web application built for sharing informational articles. It uses Web Components for modularity, modern CSS for a premium visual experience, and follows the Baseline web standards.

## Project Outline
- **Entry Point:** `index.html` (Main layout and container)
- **Styling:** `style.css` (Modern CSS, Responsive, Premium feel)
- **Logic:** `main.js` (Web Components: `<blog-header>`, `<blog-post-card>`, `<blog-footer>`)
- **Features:**
    - Google AdSense integration.
    - `ads.txt` authorization.
    - AdSense account verification meta tag.
    - Responsive article feed.
    - Modular component architecture.
- **Environment:** Firebase Studio / Code OSS
- **Version Control:** Git (GitHub: https://github.com/Longhayoung/product_builder)

## Current Change Plan: Blog Transformation
- **Objective:** Create a premium-looking blog website for information sharing.
- **Implementation Steps:**
    1. **Theming & Variables:** Update `style.css` with a vibrant color palette, modern typography, and layout variables.
    2. **Component Development (JS):** Define `<blog-header>`, `<blog-post-card>`, and `<blog-footer>` as custom elements.
    3. **Content Structure:** Use an array of article objects in `main.js` to dynamically populate the feed.
    4. **Layout (HTML/CSS):** Structure `index.html` with components and implement a responsive feed.
    5. **Aesthetics:** Add textures, deep shadows, and interactive effects.
- **Verification:**
    - Ensure all components render correctly.
    - Verify mobile responsiveness.
    - Check for console errors.
